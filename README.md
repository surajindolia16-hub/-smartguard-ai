# 🛡️ SmartGuard AI — Content Moderation System

## 🚀 Overview
SmartGuard AI is a real-time content moderation system that analyzes text messages and classifies them into SAFE, FLAGGED, or TOXIC categories.

The system is designed to be **simple, explainable, and production-like**, making it suitable for real-world social platforms.

---

## 🎯 Problem Statement
Social media platforms face challenges in detecting toxic and harmful content in real-time while maintaining transparency in moderation decisions.

---

## 💡 Solution
SmartGuard AI solves this by:
- Detecting harmful words in text
- Classifying content into clear categories
- Providing explainable moderation reasons
- Showing live moderation feed dashboard

---

## ⚙️ Tech Stack
- Frontend: React.js
- Backend: Node.js + Express
- Styling: Inline CSS (simple dashboard UI)
- Communication: REST API

---

## 🧠 How It Works
1. User text is sent to backend API
2. Backend checks for harmful words
3. System assigns label:
   - SAFE
   - FLAGGED
   - TOXIC
4. Response is sent back to frontend
5. Live dashboard updates instantly

---

## 📊 Features
- Real-time moderation simulation
- Explainable AI decisions
- Live feed dashboard
- Simple and clean UI
- Easy to scale to real ML models
